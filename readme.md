Follow the steps
1.Clone the repo
```
git clone https://github.com/umeshlab/terminal_games.git
```
2.Install the requirements
```
pip install -r requirements.txt
```

3.Go to the public directory
```
cd public
```
4.Run the main.py file
```
python main.py
    or
py main.py
```
if your using linux
```
python3 main.py
```
